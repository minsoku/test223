# VR 방탈출 게임

메타퀘스트3와 컨트롤러를 사용하여 플레이할 수 있는 WebXR 기반 방탈출 게임입니다.

## 🎮 게임 개요

간단한 방 안에서 열쇠를 찾아 문을 열고 탈출하는 VR 방탈출 게임입니다.

### 게임 요소
- **상자**: 열쇠가 숨겨져 있는 상자
- **황금 열쇠**: 문을 열기 위해 필요한 열쇠
- **문**: 열쇠로 열 수 있는 탈출구

## 🕹️ 조작법

### VR 컨트롤러
- **트리거 버튼**: 오브젝트 선택 및 상호작용
- **컨트롤러 포인팅**: 상호작용할 오브젝트를 가리키기

### 게임 진행
1. VR 헤드셋을 착용하고 "VR 모드 시작" 버튼 클릭
2. 방 안을 둘러보며 상자를 찾기
3. 컨트롤러로 상자를 클릭하여 열기
4. 나타난 황금 열쇠를 클릭하여 획득
5. 문으로 가서 클릭하여 열기
6. 방탈출 성공! 🎉

## 🛠️ 설치 및 실행

### 필요 요구사항
- **VR 헤드셋**: Meta Quest 3 (또는 WebXR 지원 헤드셋)
- **브라우저**: Chrome, Edge, Firefox (WebXR 지원)
- **HTTPS**: WebXR은 HTTPS에서만 작동합니다

### 로컬 실행
1. 프로젝트 파일을 다운로드합니다
2. 로컬 웹 서버를 실행합니다:
   ```bash
   # Python 3.x
   python -m http.server 8000
   
   # 또는 Node.js
   npx http-server
   
   # 또는 Live Server (VS Code 확장)
   ```
3. HTTPS를 활성화합니다 (필요시):
   ```bash
   # Python HTTPS 서버
   python -m http.server 8000 --bind 127.0.0.1
   ```
4. 브라우저에서 `https://localhost:8000` 접속

### VR 헤드셋 설정
1. Meta Quest 3에서 개발자 모드 활성화
2. 브라우저에서 WebXR 기능 활성화
3. 헤드셋과 컨트롤러가 정상적으로 연결되었는지 확인

## 🎯 게임 특징

- **몰입형 VR 경험**: 실제 방에 있는 듯한 3D 환경
- **직관적인 조작**: 컨트롤러로 자연스러운 상호작용
- **부드러운 애니메이션**: 상자 열기, 문 열기 등 부드러운 애니메이션
- **실시간 메시지**: VR 공간에서 실시간 메시지 표시
- **그림자 효과**: 실감나는 조명과 그림자 시스템

## 🐛 문제 해결

### VR 모드가 시작되지 않는 경우
1. HTTPS로 접속했는지 확인
2. VR 헤드셋이 정상적으로 연결되었는지 확인
3. 브라우저의 WebXR 지원 여부 확인
4. 개발자 모드가 활성화되었는지 확인

### 컨트롤러가 인식되지 않는 경우
1. 컨트롤러 배터리 확인
2. 헤드셋과 컨트롤러 재연결
3. 브라우저 새로고침

### 성능 문제
1. 브라우저의 하드웨어 가속 활성화
2. 다른 탭 및 프로그램 종료
3. VR 헤드셋의 그래픽 설정 조정
